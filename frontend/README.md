# ProTalent UI 🚀

Interface web do **ProTalent**, uma plataforma focada na criação, edição e gerenciamento de currículos profissionais modernos, responsivos e otimizados para o mercado de trabalho.

## ✨ Funcionalidades

- 📄 Criação e gerenciamento de currículos
- 🎨 Múltiplos layouts (Standard, Profile, Modern)
- ✏️ Edição em tempo real do conteúdo
- 🧠 Organização de experiências, educação e habilidades
- 📥 Download de currículo em PDF (em evolução)
- 🔍 Busca, paginação e status de currículos

## 🛠️ Tecnologias utilizadas

- **React**
- **Vite**
- **Tailwind CSS**
- **React Router**
- **Lucide Icons**
- **Framer Motion**
